对象与类

对象的创建

比如Student s = new Student()实例化一个对象，其实经历了如下几个过程：

  1. 将Student.class加载到内存中
  2. 在栈内存中给s开辟内存空间。
  3. 在堆内存给Student类申请一个内存空间。
  4. 给成员变量进行默认初始化，0 null false…
  5. 自定义给成员变量初始化赋值
  6. 初始化完毕，把堆内存地址赋值给栈内存的s变量
